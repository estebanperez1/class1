How do we add animations?

animations are done using css, there are a few ways to do this and I will show a couple below.

The first way is by doing a css animation the animation definition is shown below

@keyframes spin
{
    from{
        transform:translate(0px) rotate(0deg) ;
    }to {
        transform:translate(500px) rotate(360deg) ;
    }
}

Next we need to apply the css to an element. In this case I will be using a class to apply the style

.pointer {
    animation-duration : 2s ;
    animation-name : spin ;
    animation-iteration-count : infinite ;
    animation-direction : alternate ;
    height : 100px ;
    width : 100px ;
    background-color : red ;
    postition : absolute ;
}

Lastly we want to make the div that the animation gets applied to.

<div class = "pointer" ></div>

The second type of animation relies on a user for it to start.

The first thing we do is add a transition to a class/tag/id. In this example we will be using an ID

#interactiveDiv {
    transition : all 1s ;
    height : 100px ;
    width : 100px ;
    background-color : red ;
}

If any of the css for this element changes it will change over the duration of 1 second

We need to set a condition for the animation to fire. We will be using hover

#interactiveDiv:hover {
    height : 200px ;
    width : 200px ;
    background-color : blue ;
}

last thing that we need to do is make a div with the id of interactiveDiv.

<div id = "interactiveDiv" ></div>
